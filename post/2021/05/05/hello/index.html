Integrating Python code in R

May 5, 2021   

Hello! In this blog post I will discuss how R Markdown can utilize more than one language in its code chunks. You can type code in R like

#R Code Chunk 
numbers <- c(1,2,3,4,5)
mean(numbers)
## [1] 3

You can also type code in Python like

#Python Code Chunk
import statistics
from statistics import mean
numbers1 = [1,2,3,4,5]
mean(numbers1)
## 3

Reticulate lets you interact R code with Python code!

#R code chunk
library(reticulate)

hi<-"Hello"
#Python code chunk
hi="world"
print(r.hi,hi)
